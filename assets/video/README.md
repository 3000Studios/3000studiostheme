# Video Assets

This directory contains video assets for the theme.

## nav-bg.mp4

Navigation background video. The theme will gracefully handle if this file is missing.

To add your own video:
1. Place your video file here as `nav-bg.mp4`
2. Recommended format: MP4 (H.264)
3. Recommended size: 1920x1080 or smaller
4. Keep file size under 5MB for better performance

## Notes

The theme is designed to work without these video files - they are optional enhancements.
If a video is not found, the theme will display the background without it.
